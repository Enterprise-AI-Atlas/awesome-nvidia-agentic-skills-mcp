# AGENTS.md — Awesome NVIDIA Agentic Skills & MCP Servers

## Purpose

This repository is a curated registry of agentic skills, MCP servers, and coding-agent integrations for NVIDIA products. It is intended for both human developers and automated agents.

## Scope

Cover resources that help AI agents, coding assistants, and MCP clients interact with NVIDIA products:

- Agentic skills for NVIDIA products (OpenCode skills, Cursor rules, Claude Code instructions, etc.).
- MCP servers that expose NVIDIA products or APIs.
- Provider extensions or configurations that connect coding agents to NVIDIA endpoints.
- Official reference implementations with clear agent-facing interfaces.

## Style

- Follow the same format as the other `awesome-*` repositories in the Enterprise AI Atlas organization.
- Label official NVIDIA resources as `Official` and third-party resources as `Community`.
- Prefer official documentation and GitHub repository links.

## Maintenance

- Run `./scripts/validate-links.sh` before finishing any edit.
- Place entries under the NVIDIA product line they primarily target.

## Important reminders

- Do not provide legal advice.
- Verify that links resolve and are current.
